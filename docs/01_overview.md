# HOBBS - プロジェクト概要書

## 1. プロジェクト概要

**HOBBS** (Hobbyist Bulletin Board System) は、Telnetプロトコルで接続する昔ながらのパソコン通信BBSホスト局プログラムです。1980〜90年代に栄えたパソコン通信文化を現代の技術で再現し、レトロなテキストベースのコミュニケーション体験を提供します。

### プロジェクト名の由来
- **HOBBS** = **Hob**byist **B**ulletin Board **S**ystem
- 趣味人・愛好家のためのBBSという意味を込めています

## 2. 目的・背景

### 目的
- テキストベースの懐かしいBBS体験を現代に蘇らせる
- シンプルで堅牢な小規模コミュニティプラットフォームの提供
- レトロコンピューティング愛好家のための実用的なBBSホスト局の実現

### 背景
パソコン通信時代のBBS（草の根BBS、商用BBSなど）は、インターネット普及以前の重要なコミュニケーション手段でした。現代のWebベースのSNSとは異なる、テキストのみのシンプルなインターフェースには独特の魅力があります。HOBBSはその体験を、現代の安全な技術基盤の上で再現することを目指します。

## 3. 対象ユーザー

- レトロコンピューティング愛好家
- テキストベースUIに親しみのある方
- 小規模な趣味コミュニティを運営したい方
- 昔のパソコン通信を懐かしむ方
- シンプルなコミュニケーションツールを求める方

## 4. 主要機能

| 機能 | 概要 |
|------|------|
| 会員管理 | 入会・退会・会員情報管理・3段階権限システム |
| 掲示板 | スレッド形式/フラット形式両対応、動的追加・削除 |
| チャット | 単一ロビー形式のリアルタイムチャット |
| メール | BBS内部のプライベートメッセージ機能 |
| ファイル | フォルダ単位の権限設定可能なファイル共有 |
| ゲストアクセス | 会員登録なしでの一部コンテンツ閲覧 |

## 5. 技術仕様概要

| 項目 | 仕様 |
|------|------|
| 通信プロトコル | Telnet |
| 文字コード | ShiftJIS / UTF-8 / CP437 / PETSCII（端末プロファイルに応じて選択） |
| 画面装飾 | ANSIエスケープシーケンス対応（出力モードで制御可能） |
| 実装言語 | Rust |
| データベース | SQLite / PostgreSQL（選択可能） |
| 想定規模 | 同時接続〜10人程度 |

## 6. スコープ（実装範囲）

### 対象
- Telnetサーバとしての基本機能
- 複数エンコーディング対応（ShiftJIS / UTF-8 / CP437 / PETSCII）
- ANSIエスケープシーケンスによる画面装飾（Plain/PetsciiCtrl出力モードにも対応）
- 会員登録・認証・セッション管理
- 3段階権限システム（SysOp/SubOp/一般）
- ゲストアクセス機能
- 掲示板機能（スレッド形式/フラット形式両対応）
- チャットルーム（単一ロビー）
- BBS内部メール機能
- ファイルアップロード・ダウンロード機能
- 管理者向け設定・管理機能
- 国際化対応（日本語・英語の2言語）

### 非対象（将来検討）
- 外部メールサーバ連携（SMTP/POP3）
- 複数チャットルーム
- プライベートチャットルーム
- Webインターフェース
- モデム接続対応
- FIDO/FidoNet対応
- 多言語対応の拡張（日英以外の言語追加）
- クラスタリング・分散構成

## 7. 用語定義

| 用語 | 説明 |
|------|------|
| SysOp | System Operator。システム管理者。全権限を持つ |
| SubOp | Sub Operator。副管理者。一部管理権限を持つ |
| ゲスト | 未登録ユーザー。閲覧のみ可能 |
| スレッド形式 | トピックごとにレスがツリー状につく掲示板形式 |
| フラット形式 | 時系列に記事が並ぶ従来のBBS形式 |
| ANSI | 端末の色や装飾を制御するエスケープシーケンス |
| ShiftJIS | 日本語文字コードの一種。MS-DOSやWindowsで広く使用 |

## 8. 成功基準

- Telnetクライアントから接続し、日本語の読み書きができる
- 会員登録・ログイン・ログアウトが正常に動作する
- 掲示板への投稿・閲覧が可能
- チャットでリアルタイムに会話できる
- メールの送受信ができる
- ファイルのアップロード・ダウンロードができる
- ANSIエスケープシーケンスで画面装飾が表示される
- 10人程度の同時接続に耐えられる
